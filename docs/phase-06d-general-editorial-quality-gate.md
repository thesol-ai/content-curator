# Phase 6D - General editorial quality gate

This phase replaces brittle one-off source fixes with a more general editorial quality layer.

It adds generic rejection for low-utility marketing, private-asset access, evergreen interviews, soft speculation, and project updates without material market, security, regulatory, or user-impact signals.

It also adds broad low-quality query exclusions to crypto Apify rotation queries so the pipeline pulls less garbage before AI scoring.
